# 贡献指南

感谢你对本项目的关注！本项目是一个学习性质的爬虫项目，欢迎提交改进建议。

## 如何贡献

### 提交 Issue

如果你发现了问题，请提交 Issue 并包含以下信息：
- 问题的详细描述
- 复现步骤
- 预期行为 vs 实际行为
- 环境信息（Python 版本、操作系统等）
- 相关的错误日志或截图

### 提交 Pull Request

1. Fork 本仓库
2. 创建你的特性分支 (`git checkout -b feature/AmazingFeature`)
3. 提交你的更改 (`git commit -m 'Add some AmazingFeature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 打开一个 Pull Request

## 代码规范

- 遵循 PEP 8 编码规范
- 添加必要的注释和文档字符串
- 确保代码可以正常运行
- 不要提交敏感信息（如真实 Cookie）

## 注意事项

- 请勿提交包含真实 Cookie 的代码
- 请确保你的修改不会增加对目标网站的请求压力
- 保持代码的学习性质，不要添加商业用途的功能
